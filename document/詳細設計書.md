# 詳細設計書（Detailed Design）

## 1. 主要モジュール・関数仕様（JavaScript）

### 1.1 初期化処理 (`window.onload`)
* **処理内容:** `localStorage` から `learning_quest_data` を取得。
* **分岐条件:** 
  * データが存在する場合: `appState` にパースして格納し、`changeScreen('menu')` を呼ぶ。
  * データが存在しない場合: `changeScreen('setup')` を呼ぶ。

### 1.2 カリキュラム自動判定関数 (`judgeCurriculum()`)
* **引数:** なし
* **戻り値:** `{ grade: 数値, term: 数値, label: 文字列 }`
* **詳細ロジック:**
  1. `appState.admissionDate` と現在のシステム日付（`new Date()`）を比較。
  2. 経過年数から「学年」を算出（4月始まり基準。1月〜3月は前年度の年数を維持）。
  3. 現在の「月」から学期を判定（4〜7月=1学期、8〜12月=2学期、1〜3月=3学期）。

### 1.3 出題処理関数 (`nextQuestion()`)
* **処理内容:** 
  1. `judgeCurriculum()` の結果から該当する学年・学期のオブジェクトを `questionBank` から参照。
  2. 選択された教科（`currentBattle.subject`）および難易度（`appState.currentDifficulty`）の問題プール（配列）を取得。
  3. `currentBattle.recentQuestions`（最大5件の配列）に含まれる問題文を除外（`filter`）。※候補が0になった場合は履歴配列を空にする。
  4. 残った配列から `Math.floor(Math.random() * length)` で1問を決定。
  5. 決定した問題文を `recentQuestions` の末尾に追加。
  6. 4つの選択肢（`c`）をランダムにシャッフルし、画面上のボタンにバインド。

### 1.4 回答判定関数 (`selectAnswer(chosen)`)
* **引数:** `chosen`（選択された選択肢の文字列）
* **処理内容:**
  * **正解の場合（`chosen === currentBattle.currentAnswer`）:**
    * 敵HPを減算（`かんたん` は34、`ふつう` は25、`むずかしい` は25×2=50）。
    * 敵要素に `damage-flash` クラスを一瞬付与して点滅させる。
    * `appState.stats[subject]` を +1 する。
  * **不正解の場合:**
    * プレイヤーHPを減算（`かんたん` は0、`ふつう` は20、`むずかしい` は34）。
    * プレイヤー要素に `damage-flash` クラスを一瞬付与して点滅させる。
  * **勝敗チェック:**
    * 敵HP <= 0: スコアを+10し、レベル計算をしてメニュー画面へ戻る。
    * プレイヤーHP <= 0: ゲームオーバーのアラートを出し、メニュー画面へ戻る。
    * 決着がつかない場合: `nextQuestion()` を再帰呼び出し。

## 2. データベース構造（問題バンク定義）
問題データは `questionBank` オブジェクトとして多階層で定義する。

```javascript
const questionBank = {
    [学年]: {
        [学期]: {
            math: {
                easy: [ { q: "問題文", a: "正解", c: ["正解", "誤答1", "誤答2", "誤答3"] } ],
                normal: [ /* ... */ ],
                hard: [ /* ... */ ]
            },
            japanese: { /* ... */ },
            life: { /* ... */ }
        }
    }
};
```

## 3. スタイルシート設計（CSSレイアウト）
* **Viewport固定:** `maximum-scale=1.0, user-scalable=no` を指定し、スマホでのピンチイン・アウトを禁止。
* **画面表示切替:** すべての `.screen` を `display: none;` とし、`.active` クラスが付与された画面のみ `display: flex;` で縦並びに配置する。
* **ダメージ演出:** `@keyframes flash` を用いて、`opacity` を 1.0 ↔ 0.3 の間で3回高速往復させる点滅アニメーションを定義。
